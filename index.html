Community contribution

Submit a dataset

Uploaded datasets are stored as pending review. They become public only after an administrator approves them.

Sign in required

Sign in with Google or GitHub before submitting a dataset.

Choose your public username

This username is shown as the dataset contributor. Your email stays private unless you choose to show it.

Accepted files: CSV, XLSX, XLS, ZIP, SQLite DB, PDF, TXT, MD. Maximum size: 50 MB per file and 2 GB total per submission.

Select the main dataset file

The main file must be CSV, XLSX, XLS, SQLite DB, or ZIP. PDF/TXT/MD are treated as supporting documentation.

Accepted images: PNG, JPG, JPEG, WEBP. Maximum size: 5 MB. This image will be shown on the dataset card after approval.
